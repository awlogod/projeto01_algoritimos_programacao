### [Contexto Jogo]

Se trata de um jogo da forca normal, com estética parecida com o jogo *Termo*. O objetivo é acertar a palavra, a partir das dicas antes que o boneco alcance o corpo completo.
Diferente do jogo da forca normal, este tem o tema de `empresários` famosos e bem sucedidos.

### [Como jogar]

*1.* Aperte `1` para iniciar o jogo;
*2.* Ao iniciar o jogo, será sorteado uma palavra;
*3.* Você terá 8 tentativas para acertar;
*4.* Caso acerte, a letra ficará `verde`, se acertar a letra, porém no local errado, ficará `amarelo`, e se errar, ficará `vermelha`
*5.* A cada acerto você ganha 5 pontos e ao errar você perde 5.
